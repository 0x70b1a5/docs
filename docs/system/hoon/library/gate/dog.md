### `++dog`

`.` optional gap

Dot followed by an optional gap, used with numbers.

Source
------

    ++  dog  ;~(plug dot gay)                               ::

Examples
--------

    /~zod/try=> 1.234.
                703
    1.234.703
    ~zod/try=> (scan "a.        " ;~(pfix alf dog))
    [~~~. ~]
