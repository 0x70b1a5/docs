### `++tine`

Switch statement cases

In [`?-`](), for example, your list of cases is a `++tine`.

Source
------

        ++  tine  (list ,[p=tile q=twig])                       ::

Examples
--------

    ~zod/try=> (ream '?-(!! @ |, ^ &)')
    [ %wthz
      p=[%.n p=~ q=[%zpzp ~]]
        q
      ~[
        [p=[%axil p=[%atom p=~.]] q=[%dtzy p=%f q=1]]
        [p=[%axil p=%cell] q=[%dtzy p=%f q=0]]
      ]
    ]
    ~zod/try=> (tine |2:(ream '?-(!! @ |, ^ &)'))
    ~[
      [p=[%axil p=[%atom p=~.]] q=[%dtzy p=%f q=1]]
      [p=[%axil p=%cell] q=[%dtzy p=%f q=0]]
    ]
