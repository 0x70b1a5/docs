### `++tusk`

List of expressions

List of [`++twig`]()s. In [`:*`](), for example, your contents is
a `++tusk`.

Source
------

        ++  tusk  (list twig)                                   ::

Examples
--------

    ~zod/try=> (ream '[1 2 3]')
    [%cltr p=~[[%dtzy p=%ud q=1] [%dtzy p=%ud q=2] [%dtzy p=%ud q=3]]]
    ~zod/try=> (tusk +:(ream '[1 2 3]'))
    ~[[%dtzy p=%ud q=1] [%dtzy p=%ud q=2] [%dtzy p=%ud q=3]]
