### `++char`

Char

A single character. odor `@tD` designates a single unicode byte. All
parsers consume `++tape`s, which are lists of `++char`.


Source
------

    ++  char  @tD                                          ::  UTF-8 byte

Examples
--------

    ~zod/try=> *char
    ''
    ~zod/try> (char 97)
    'a'
