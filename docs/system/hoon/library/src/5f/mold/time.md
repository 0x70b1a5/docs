### `++time`

Absolute date

The `@da` odor designates an absolute date atom.

Source
------

        ++  time  ,@da                                          ::  galactic time

Examples
--------

See also: [`++date`](), [odor reference]()

    ~zod/try=> `time`-<-
    ~2014.9.25..20.01.47..eeac
    ~zod/try=> :: the time is always in your context at -<-
    ~zod/try=> `time`~2014.1.1
    ~2014.1.1
