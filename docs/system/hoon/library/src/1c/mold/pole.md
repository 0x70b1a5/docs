### `++pole`

Faceless list

A `++list` without the faces `i` and `t`.

Source
------

    ++  pole  |*  a/$-(* *)                                 ::  nameless list
              $@($~ {a (pole a)})                           ::



Examples
--------

    ~zod/try=> `(pole char)`"asdf"
    [~~a [~~s [~~d [~~f ~]]]]
