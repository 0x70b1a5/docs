### `++qeu`

Queue

mold generator. An ordered [treap](http://en.wikipedia.org/wiki/Treap) of
items.

Source
------

    ++  qeu  |*  a/$-(* *)                                  ::  queue
             $@($~ {n/a l/(qeu a) r/(qeu a)})               ::


Examples
--------

    ~zod/try=> (qeu time)
    <1.qyo [* <1.sxx [a=<1.ebd [* <101.jzo 1.ypj %164>]> <101.jzo 1.ypj %164>]>]>
    ~zod/try=> (~(gas to *(qeu time)) [~2014.1.1 ~2014.1.2 ~])
    {~2014.1.2 ~2014.1.1}
