### `++aura`

'type' of atom

By convention, a short name for a category of atom. `++aura` is
circularly defined, with [`@ta`]() being the `++aura` of the ASCII subset
commonly used in urbit.

Source
------

    ++  aura  ,@ta                                          ::  atom format

Examples
--------

See also: [`++base`](), [aura reference]()

    ~zod/try=> `aura`%ux
    ~.ux
