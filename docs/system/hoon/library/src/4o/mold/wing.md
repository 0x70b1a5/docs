### `++wing`

    ++  wing  (list limb)                                   ::

Address in subject. A `++wing` is a path to a value in the subject. A
term alone is the trivial case of a `++wing`.

See also: `++twig`

    ~zod/try=> (ream 'a.+.c')
    [%cnzz p=~[%a [%.y p=3] %c]]
    ~zod/try=> (wing +:(ream 'a.+.c'))
    ~[%a [%.y p=3] %c]
