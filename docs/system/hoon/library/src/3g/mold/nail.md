### `++nail`

Location, remainder of parsed text

Indicates parsing position `p`, and remaining text to be parsed `q`.

Source
------

        ++  nail  ,[p=hair q=tape]                              ::  parsing input

Examples
--------

    ~zod/try=> +<:;~(plug cab cab)
    c=tub=[p=[p=0 q=0] q=""]
    ~zod/try=> :: tub is a ++nail
