### `++dime`

Odor-atom pair

Used in [`++coin`](). Convenience methods [`++scot`]() and [`++scow`]() (in
Section 2eL) print dimes as [`++cord`]() and [`++tape`](), respectively.
[`++slat`](), [`++slav`](), and [`++slaw`]() are used to parse atoms of
specific [odor]()s.

Source
------

    ++  dime  ,[p=@ta q=@]                                  ::

Examples
--------

    ~zod/try=> +>:(slay '0x123')
    p=[p=~.ux q=291]
