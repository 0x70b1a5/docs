### `++gon`

Long numbers

Parse long numbers - Numbers which wrap around the shell with the line

Source
------

break     ++  gon  ;~(pose ;~(plug bas gay fas) (easy ~))         ::  long numbers \ /

Examples
--------
characters bas and fas.

        ~zod/try=> (scan "\\/" gon)
        [~~~5c. ~ ~~~2f.]
        ~zod/try=> (gon [[1 1] "\\/"])
        [p=[p=1 q=3] q=[~ u=[p=[~~~5c. ~ ~~~2f.] q=[p=[p=1 q=3] q=""]]]]
