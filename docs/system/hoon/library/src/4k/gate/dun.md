### `++dun`

`--` to `~`

Parse phep, `--`, to null, `~`.

Source
------

    ++  dun  (cold ~ ;~(plug hep hep))                      ::  -- (phep) to ~

Examples
--------

    ~zod/try=> (scan "--" dun)
    ~
    ~zod/try=> (dun [[1 1] "--"])
    [p=[p=1 q=3] q=[~ u=[p=~ q=[p=[p=1 q=3] q=""]]]]
