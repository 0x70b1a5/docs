### `++soz`

Delimiting `'''`

Parse a triple-single quote, used for multiline strings.

Source
------

    ++  soz  ;~(plug soq soq soq)                          ::  delimiting '''

Examples
--------

    ~zod/try=> (scan "'''" soz)
    [~~~27. ~~~27. ~~~27.]
    ~zod/try=> (rash '"""' soz)
    ! {1 1}
    ! exit
