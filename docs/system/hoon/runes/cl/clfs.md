`:/ colfas`
===========

String to XML XX

`:/`, `colfas`, is a synthetic rune that produces `[%$ [%$ p ~] ~]`,
i.e., `[0 [0 p 0] 0]`.

Examples:

    ~zod/try=> :/(20)
    [[%~. [%~. 20] ~] ~]
    ~zod/try=> :/(add 2 2)
    [[%~. [%~. 4] ~] ~]

Wraps `++twig` in `[%$ [%$ .] ~]~`, used for interpolation.
