# `:zinc`, `^&`, "ketpam", `{$zinc p/twig}`

Convert `%gold` to `%zinc`.

Converts a `%gold` core `p` into a `%zinc` core.

Examples:

    ~zod/try=> |=(@ 1)
    < 1.odf
      {@ {our/@p now/@da eny/@uvI} $~ <282.ucw 42.juj 402.arm 110.jyx 1.ztu $151>}
    >
    ~zod/try=> ^&(|=(@ 1))
    < 1&odf
      {@ {our/@p now/@da eny/@uvI} $~ <282.ucw 42.juj 402.arm 110.jyx 1.ztu $151>}
    >
