### `++gate`

Function

A core with one arm, `$`--the empty name--which transforms a sample noun into a product
noun. If used dryly as a type, the subject must have a sample type of `*`.


Source
------

    ++  gate  $-(* *)                                       ::  general gate


Examples
--------

See also: `++lift`, `++cork`

    > ? gate
      <1.ybc {* <110.jyx 1.ztu $151>}>
    <1.ybc {* <110.jyx 1.ztu $151>}>

    > (`gate`|=(a/* [a 'b']) 1)
    [1 'b']
