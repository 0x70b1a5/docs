# `:code`, `!=`, "zaptis" `$code p/twig}`

Show nock.

`!=` is a natural rune that produces the formula of `p` as a Nock noun.

Regular form: *1-fixed*

Examples:

    ~zod:dojo> !=(20)
    [1 20]
    ~zod:dojo> !=(~zod)
    [1 0]
    ~zod:dojo> !=((add 2 2))
    [8 [9 3.110.356 0 31] 9 2 [0 4] [7 [0 3] 1 2 2] 0 11]
    ~zod:dojo> !=(|=(a=@ [a a]))
    [8 [1 0] [1 [0 6] 0 6] 0 1]
    ~zod:dojo> !=(+(2))
    [4 1 2]
    ~zod:dojo> !=(.?(2))
    [3 1 2]
    ~zod:dojo> !=(.*(~ 2))
    [2 [1 0] 1 2]
    ~zod:dojo> !=(2)
    [1 2]
    ~zod:dojo> !=(.)
    [0 1]
    ~zod:dojo> !=(!!)
    [0 0]
