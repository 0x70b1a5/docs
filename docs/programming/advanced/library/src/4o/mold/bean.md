### `++bean`

Boolean

`0`, `&`, or `%.y` are true, and `1`, `|`, and `%.n` are false.

Source
------

    ++  bean  ,?                                            ::  0=&=yes, 1=|=no

Examples
--------

    ~zod/try=> *bean
    %.y

    ~zod/try=> `bean`&
    %.y
    ~zod/try=> `bean`|
    %.n
