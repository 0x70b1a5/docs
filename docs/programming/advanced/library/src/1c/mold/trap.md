### `++trap`

Core with one arm `$`

A trap is a core with one arm `++$`.

Source
------

    ++  trap  |*(a/_* _|?(*a))                        ::  makes perfect sense


Examples
--------

    ~zod/try=> *trap
    <1.mws 101.jzo 1.ypj %164>
    ~zod/try=> (*trap)
    0
    ~zod/try=> (|.(42))
